## Ye Olde Contact List App

### To Spin Up The App:

```sh
git clone git@github.com:stephenkarpeles/contact-list.git
```

```sh
cd contact-list
```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
